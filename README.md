# Doctor Appointment Booking System

A Django web application that allows patients to book appointments with doctors.

## Features

- User registration
- Doctor and patient roles
- Book appointments
- Doctor dashboard to view appointments
- Cancel appointment option

## Technologies Used

- Python
- Django
- HTML
- Bootstrap
- SQLite

## How to Run the Project

1. Clone the repository

git clone https://github.com/yourusername/doctor-appointment-booking-system.git

2. Go to the project folder

cd doctor-appointment-booking-system

3. Create virtual environment

python -m venv venv

4. Activate virtual environment

venv\Scripts\activate

5. Install Django

pip install django

6. Run migrations

python manage.py migrate

7. Start server

python manage.py runserver
